

### Machine Learning Directory

This directory contains code for a variety of data operations, such as Logistic Regression, Linear Regression, and Exploratory Data Analysis (EDA). Currently, the directory is divided into three folders:

*Logistic Regression: This folder contains Logistic Regression code.

*Linear Regression: The code for Linear Regression operations can be found in this folder.

*Exploratory Data Analysis (EDA): This folder contains codes for exploratory data analysis
